# Keil Studio examples

The following repositories contain meaningful example
projects for the [Keil Studio](https://mdk-packs.github.io/vscode-cmsis-solution-docs/index.html) IDE
([CMSIS solution-based](https://open-cmsis-pack.github.io/cmsis-toolbox/YML-Input-Format/)).

| Repository | Hardware used | Content |
|------------|---------------|---------|
| [Blinky_SAMD21_Xplained_Pro](https://github.com/KeilChris/Blinky_SAMD21_Xplained_Pro) | [Microchip SAMD21 Xplained Pro](https://www.microchip.com/en-us/development-tool/atsamd21-xpro) | Simple blinky example; debug via integrated Atmel EDBG (CMSIS-DAP v1.x) |
| [MC00_Curiosity_Pro_Blinky](https://github.com/KeilChris/MC00_Curiosity_Pro_Blinky) | [Microchip PIC32CM MC00 Curiosity Pro](https://www.microchip.com/en-us/development-tool/ev15n46a) | Simple blinky example; debug via integrated Atmel EDBG (CMSIS-DAP v1.x) |
| [Blinky_XMC2Go](https://github.com/KeilChris/Blinky_XMC2Go) | [Infineon XMC2Go](https://www.keil.arm.com/boards/infineon-xmc-2go-v1-d3962c0/projects/) | Simple blinky example; debug via integrated J-Link Lite. Using semihosting for `printf` debug output |
| [Blinky_FRDM-MCXN947](https://github.com/KeilChris/Blinky_FRDM-MCXN947) | [NXP FRDM-MCXN947](https://www.keil.arm.com/boards/nxp-frdm-mcxn947-634a158/projects/) | Simple blinky example; debug via integrated MCU-Link (CMSIS-DAP v2.x) |
| [Blinky_LPCXpresso55S69](https://github.com/KeilChris/Blinky_LPCXpresso55S69) | [NXP LPCXpresso55S69](https://www.keil.arm.com/boards/nxp-lpcxpresso55s69-56e55af/projects/) | Simple blinky example; debug via integrated LPC-Link (CMSIS-DAP v2.x) |
