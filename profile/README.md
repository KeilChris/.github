# About me

I am the product manager for the Arm Keil tools, working in Arm's Munich office. Here, I store meaningful example
projects for the [Keil Studio](https://mdk-packs.github.io/vscode-cmsis-solution-docs/index.html) IDE
([CMSIS solution-based](https://open-cmsis-pack.github.io/cmsis-toolbox/YML-Input-Format/)).

## Keil Studio examples

The following repositories contain ready-to-run examples:

| Repository | Hardware used | Content |
|------------|---------------|---------|
| [Blinky_SAMD21_Xplained_Pro](https://github.com/KeilChris/Blinky_SAMD21_Xplained_Pro) | Microchip SAMD21 Xplained Pro | Simple blinky example; debug via integrated Atmel EDBG (CMSIS-DAP v1.x) |
| [MC00_Curiosity_Pro_Blinky](https://github.com/KeilChris/MC00_Curiosity_Pro_Blinky) | Microchip PIC32CM MC00 Curiosity Pro | Simple blinky example; debug via integrated Atmel EDBG (CMSIS-DAP v1.x) |
| [Blinky_XMC2Go](https://github.com/KeilChris/Blinky_XMC2Go) | Infineon XMC2Go | Simple blinky example; debug via integrated J-Link Lite. Using semihosting for `printf` debug output |
| [Blinky_FRDM-MCXN947](https://github.com/KeilChris/Blinky_FRDM-MCXN947) | NXP FRDM-MCXN947 | Simple blinky example; debug via integrated MCU-Link (CMSIS-DAP v2.x) |
| [Blinky_LPCXpresso55S69](https://github.com/KeilChris/Blinky_LPCXpresso55S69) | NXP LPCXpresso55S69 | Simple blinky example; debug via integrated LPC-Link (CMSIS-DAP v2.x) |
